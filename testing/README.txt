Soccer MCP Testing Script

Usage:
  python soccer_test_simple.py

The script will guide you through:
1. Choose a league (1-6)
2. Choose finished or upcoming games  
3. Select a specific game from the list
4. View game details (scores, stats, etc.)
5. Optionally view top scorers for the league

Features:
- Limited output to prevent crashes
- Interactive menu system
- Works with live soccer MCP server
- Shows recent finished games or upcoming games

Note: Make sure your .env file is set up in the parent directory with any required API keys.
